How I set up my project "Into My Universe".

From concept to production

Posted by John Cvn on August 24, 2019

Thanks to take time to read my post, I appreciate, really ! This post will be more serious than my first one because I will explain you how I set up my project, which softwares, which services I'm using. Everything about the project (front-end and back-end).

I wanted to take seriously this project, so I decided, before to build my website, to use which services I will use. I wanted to organize myself seriously and properly.

Softwares and Services

I will just talking about services I'm using for versionning and productivity. I will not talking about IDE because it's personnal and not really interesting.
So, firstly and obviously, I'm using GitHub for the versionning of my website (I'm not learning you what is GitHub...). I can work cleanly on my project without do huge mistakes and it's because I like the concept of open source.
Then, I wanted to be organize and professional with my project, so it's for that I'm using Trello.
Trello is a web-based list-making application to manage your projects. I decide to set some boards in public to let you the possibilities to see how I'm organized myself and to follow the website updates.
